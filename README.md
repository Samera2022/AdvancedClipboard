## Features

*   **Clipboard History**: Automatically saves your clipboard history, supporting both text and images. Never lose a copied item again.
*   **Quick Access**: Use global hotkeys (`Ctrl + Up` / `Ctrl + Down`) to instantly bring up the history window and navigate through your clips.
*   **Restore & Paste**: Select any item from your history to restore it to the clipboard and paste it immediately.
*   **System Tray Integration**: Runs quietly in the background with a system tray icon for easy access to settings and logs.
*   **Lightweight**: Built with native C++ and Win32 API for maximum performance and minimal resource usage.

## Getting Started

### Prerequisites
*   Windows 10 or later.
*   [Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170) (if not already installed).

### Installation
1.  Download the latest release from the [Releases](https://github.com/Samera2022/AdvancedClipboard/releases) page.
2Run `AdvancedClipboard.exe`.

### Usage
1.  **Copy**: Copy text or images as you normally would (`Ctrl + C`).
2.  **Retrieve**: Press `Ctrl + Up` or `Ctrl + Down` to open the clipboard history popup.
3.  **Select**: Use the arrow keys to navigate the list.
4.  **Paste**: Press `Enter` to select the item. It will be copied to your clipboard and ready to paste.
